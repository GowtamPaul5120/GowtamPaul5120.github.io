# Using a Resource in a Workflow

To use a resource in a workflow

1. Open the workflow and drag an activity (such as StreamReader, REST Client) on to the canvas.
2. In the activity settings, select the resource from the **Resource Dropdown**.
3. Configure any additional parameters as needed.
4. Save the workflow and test.

## **Best Practices for Using Resources**

* **Descriptive Names**: Use clear names to identify resource purposes easily.
* **Validate Imports**: Ensure imported resources match your environment.
* **Backup Resources**: Export important resources regularly for recovery.
