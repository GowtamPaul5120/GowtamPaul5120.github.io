You create resources within an application. Once the resource is created, it's available to all workflows in that application.

## Creating Resources

To add a resource to an application:

1. Open the application, right-click the **Resources** folder and select **Add Resource**.\
   You can also click on the application in the Applications panel and, in the Create page that appears on the right, click **Resource**.\
   \
   The Configure Resource modal appears.
2. Choose the resource type you want to create, or search for an existing one, click the resource type you want, and click **Save**.\
   The resource is now created and saved in the Resources folder.
3. Add and save details associated with the resource as appropriate.\
   Your resource is now ready for use in workflows associated with the parent application.

## Renaming Resources

To rename a resource:

1. Go to the **Resources** section.
2. Click the **ellipsis (⋮)** next to the resource you want to rename.
3. Select **Rename**.
4. Enter the new name and press **Enter**.

You can also open the resource and update all of its fields, including its name.
