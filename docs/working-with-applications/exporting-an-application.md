# Exporting an Application

Exporting an application creates a backup of its workflows, resources, schemas, and configurations.

To export an application

- Right-click on the application and select **Export Application**.
- Save the file to your system for backup or reuse.

![Export Application animation](../../assets/images/Export-Application (1).gif)

:::info
Exported files ensure that applications can be restored or shared with other environments seamlessly.
:::
