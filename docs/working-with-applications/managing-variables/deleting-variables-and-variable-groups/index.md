## Deleting Variables

To delete a variable:

1. Navigate to the variable group containing the variable that you want to delete.
2. Click the **Ellipsis** icon at the right end of the row that contains the variable and select **Delete** from the options that appear.
3. Click **Yes** in the warning message that appears.

## Deleting Variable Groups

To delete a variable group:

1. Navigate to the variable group that you want to delete.

* Click the **Ellipsis** icon at the right end of the row that contains the variable group and select **Delete** from the options that appear.
* Click **Yes** in the warning message that appears.
