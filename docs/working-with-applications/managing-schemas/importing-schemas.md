---
title: "Importing Schemas"
---


You can import schemas into your application in any of the following ways:

* **From the Schemas folder in your application.**\
  For guidance on how to perform this task, see [creating-schemas.md](/docs/working-with-applications/managing-schemas/creating-schemas).
* **From the Input tab in activities.**\
  While configuring the input fields of an activity, import a schema to define the expected data structure. For guidance on working with the Add Schema modal that appears, see [creating-schemas.md](/docs/working-with-applications/managing-schemas/creating-schemas).
*   **From the Output tab in activities.**

    While configuring the output fields of an activity, import a schema to specify the structure of data being passed forward. For guidance on working with the Add Schema modal that appears, see [creating-schemas.md](/docs/working-with-applications/managing-schemas/creating-schemas).&#x20;
