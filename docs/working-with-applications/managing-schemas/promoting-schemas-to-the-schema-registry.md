---
title: "Promoting Schemas to the Schema Registry"
---


The Schema Registry maintains a copy of finalized schemas that can be shared across your organization, ensuring that users across teams have access to an approved set of defined schemas.

To promote a schema to the Schema Registry:

1. Open the schema in View mode from your application hierarchy.
2. Click **Promote**.\
   The Promote Schema modal appears.
3. Select the Schema Registry **Collection** to which the schema must be added.
4. Optionally, update the name of the schema by clicking the **Edit** icon adjacent to the name of the schema.
5. Click **Promote**.

The schema you just promoted will now be available in the folder you selected in the Schema Registry to all users in the organization.
