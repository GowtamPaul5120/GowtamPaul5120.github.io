# Deleting an Application

:::info
Deleting an application permanently removes it along with all its workflows, resources, and schemas.
:::

To delete an application, right-click the application and select **Delete**. Confirm the action in the popup that appears by entering the name of the application you're deleting.

:::info
This action is irreversible. Any workflows or dependencies tied to the application will be lost. To avoid data loss, export the application before deleting if needed.
:::
