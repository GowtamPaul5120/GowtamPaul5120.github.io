---
title: Editing Workflow Properties
---


Use the configuration panel below the canvas to edit a workflow's properties.

To view a workflow's properties, open the workflow and click on the canvas to ensure you haven't selected any of the workflow's activities. Details associated with the open workflow appear in the configuration panel.

## Configuration

Use this tab to update the **Name** and **Description** of the open workflow.

## Attribute Tags

Use this tab to view the attribute tags used in the workflow. You can also delete the tags used, you can modify them directly from the activity.

See [Adding Attribute Tags to Activities](/docs/working-with-activities/adding-attribute-tags-to-activities) for more details.
