# Exporting a Workflow

To export a workflow:

1. Navigate to the workflow in the Applications panel.
2. Right click on the workflow and select **Export Workflow**.\
   If the workflow has no dependency on other files, the application exports it immediately to your system as a ZIP file.\
   \
   However, if there are any dependencies in the workflow, the Export Workflow modal lists out all references in the workflow, and you can select the files referenced that you want to export along with the workflow.
3. Click **Export**.
