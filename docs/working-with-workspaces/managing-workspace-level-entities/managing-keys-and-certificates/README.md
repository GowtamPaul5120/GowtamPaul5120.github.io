# Managing Keys and Certificates

Use the Settings > Workspace > Keys and Certificates page to manage encryption / authentication keys and certificates.

## Encryption and Authentication Keys

Security certificates, such as SSL, help verify the identity of websites and online services and enable encrypted connections between web browsers and web servers. Keys ensure that only authorized parties can read your data. Without the correct key and / or certificate, your data remains inaccessible and unreadable.

Koodisi supports PGP keys and SSL and SSH certificates to help you secure your operations and data.
