---
title: "Creating a Key or Certificate"
---


To create a key or certificate:

1. Click **Keys and Certs** in the Workspace section in the Settings page.\
   The Keys and Certificates page appears.
2. Click **Create** and select the type of artifact you want to create.\
   The Add Keys side-panel appears.
3. Enter a **Name** for the artifact you want to create and enter a **Description**.
4. Select the **Environments** in which you want the artifact to be available.
5. Click the **Certificate Type** field and select the type of artifact you want to create.
   1. If you select _SSL_, an additional **Certificate Subtype** field appears. Click this field and select the   certificate subtype that you want to use.&#x20;
6. Enter the **Public Key** and **Private Key** in the fields provided, or use the **Upload** button to select the required key files from your system.
7. Enter the **Passphrase** that you used to create the artifact.
8. Click **Save**.\
   Your artifact is now created, and the View Certificate side-panel displays its details. \
   You can now update only the **Description** and **Environment** details if required.&#x20;
9. Click **Save** in the View Certificate side panel.
