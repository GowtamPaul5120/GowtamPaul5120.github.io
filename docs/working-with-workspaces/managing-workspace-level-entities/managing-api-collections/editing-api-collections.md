---
title: "Editing API Collections"
sidebar_position: 2
---


To edit an API collection:

1. Navigate to the Settings > Workspace > API Collections page and click the API that you want to edit.\
   Details associated with the API appear.
2. Click the **Edit** button in the upper-right section of the page.\
   The Update Collections side panel appears.
3. Update the **Name** and **Description** of the API as appropriate.
4. Click the **Choose Paths** field and, from the drop-down list that appears, choose the deployed application or workflow that you want to add to your collection.\
   \
   When you add an API to your collection, it appears in the Choose Paths field. To remove an API from the collection, click the **X** icon adjacent to it.
5. To copy the path to an API already in your collection, hover over it and click the **Copy Path** icon that appears.
6. Click **Update**.
