---
title: "Creating API Collections"
sidebar_position: 1
---

To create an API collection:

1. Navigate to the Settings > Workspace > API Collections page and click **Create Collection**.\
   The Create Collections page appears.
2. Enter a **Name** for the API collection and, optionally, provide a **Description**.
3. Click the **Choose Paths** field and, from the drop-down list that appears, select the deployed applications or any of their workflows that you want to add to the collection as an API.\
   Do this for every deployed application or workflow that you want to add to the collection.

:::info
To copy the path to an application or workflow API that you added to the collection, hover over the API and click the **Copy Path** icon.
:::

4. Click **Create**.
