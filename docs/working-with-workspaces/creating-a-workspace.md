# Creating a Workspace

To create a workspace:

1. Navigate to the Workspaces page by either clicking on your workspace in the Koodisi header and selecting **View all workspaces**, or by clicking **View workspaces** in the Koodisi Home page.
2. Click the **Create** button on the Workspace page.\
   The Create Workspace modal appears.
3. Enter the **Name** and **Description** of the workspace.
4. Toggle **Private Workspace** to restrict access to specific members, if needed. For guidance on how to provide access to private workspaces, see [Workspace General Settings](/docs/administering-koodisi/configuring-workspace-settings/workspace-general-settings).
