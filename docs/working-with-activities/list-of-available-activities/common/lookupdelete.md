The **Lookup Delete** activity removes an entry from a lookup table.

## Configuring the Lookup Delete Activity

To configure the **Lookup Delete** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Optional | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Lookup Table | Optional | Specify the lookup table for this activity. | Lookup Table |
| Workflow Loader | Optional | Specify the workflow loader for this activity. |  |
