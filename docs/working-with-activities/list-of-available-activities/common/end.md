title: End

The **End** activity marks the end of a workflow execution path.

:::info
The End activity marks the terminal step of a workflow. No activities can follow it.
:::

## Configuring the End Activity

To configure the **End** activity, fill in the required fields in the activity panel.

### Configuration

| Field       | Required | Description                                | Example              |
| ----------- | -------- | ------------------------------------------ | -------------------- |
| Name        | Optional | Specify the name for this activity.        | MyActivity           |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |

### Input

_Map the required input fields to provide data for this activity._
