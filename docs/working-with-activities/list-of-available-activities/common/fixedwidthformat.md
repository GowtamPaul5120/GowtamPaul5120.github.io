The **Fixed Width Format** activity parses or generates fixed-width format data.

## Configuring the Fixed Width Format Activity

To configure the **Fixed Width Format** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Required | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Records | Optional | Records for this activity. |  |
