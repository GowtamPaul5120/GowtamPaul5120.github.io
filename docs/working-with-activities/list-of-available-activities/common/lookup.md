# Lookup

The **Lookup** activity retrieves a value from a lookup table by key.

## Configuring the Lookup Activity

To configure the **Lookup** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Optional | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Lookup Table | Optional | Specify the lookup table for this activity. | Lookup Table |
| Workflow Loader | Optional | Specify the workflow loader for this activity. |  |

### Input

_Map the required input fields to provide data for this activity._

### Output

_The output fields are dynamically populated based on the activity result._
