The **Update WF Variable** activity updates a workflow-level variable.

## Configuring the Update WF Variable Activity

To configure the **Update WF Variable** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Optional | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
