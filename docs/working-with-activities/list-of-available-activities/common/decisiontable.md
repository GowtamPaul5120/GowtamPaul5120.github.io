The **Decision Table** activity evaluates input data against a rules table and returns the matching output.

## Configuring the Decision Table Activity

To configure the **Decision Table** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Optional | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Decision Table | Optional | Specify the decision table for this activity. | Decision Table |

### Input

_Map the required input fields to provide data for this activity._

### Output

_The output fields are dynamically populated based on the activity result._
