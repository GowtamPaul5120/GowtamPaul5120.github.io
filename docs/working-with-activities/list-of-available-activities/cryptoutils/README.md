# CryptoUtils

Use activities in the CryptoUtils group to decrypt and verify the authenticity of content using cryptographic&#x20;methods.

* [VerifySignature](/docs/working-with-activities/list-of-available-activities/cryptoutils/verifysignature): Enables you to confirm that content hasn’t been tampered with and comes from a trusted source.
* [Decrypt](/docs/working-with-activities/list-of-available-activities/cryptoutils/decrypt): Enables you to restore encrypted data back to its original, human-readable form.
