# Data Formatter

The **Data Formatter** activity applies formatting rules to transform data fields.

## Configuring the Data Formatter Activity

To configure the **Data Formatter** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Required | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
