The **CSV Reader** activity parses CSV data and returns structured records.

## Configuring the CSV Reader Activity

To configure the **CSV Reader** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Optional | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Source Definition | Optional | Drag and drop a CSVFormat resource from the workspace. |  |
| Data Source | Optional | Drag and drop a ABSStreamReader resource from the workspace. |  |

### Input

_Map the required input fields to provide data for this activity._

### Output

_The output fields are dynamically populated based on the activity result._
