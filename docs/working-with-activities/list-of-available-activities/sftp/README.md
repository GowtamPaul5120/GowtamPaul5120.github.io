# SFTP

The SFTP activity group provides activities for interacting with SFTP servers, including reading, writing, listing, and deleting files.

Before using the SFTP activities, ensure you have configured an **SFTP Connection** resource in your workspace with valid server credentials.

## Available Activities

* [Delete SFTP File](DeleteSFTPFile.md)
* [List SFTP File](ListSFTPFile.md)
* [Read SFTP File](ReadSFTPFile.md)
* [SFTP Connection](SFTPConnection.md)
* [SFTP File Info](SFTPFileInfo.md)
* [Write SFTP File](WriteSFTPFile.md)
