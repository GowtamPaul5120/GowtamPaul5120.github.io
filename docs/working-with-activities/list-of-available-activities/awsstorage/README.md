# AWSStorage

Use AWSStorage activities to:

* [List all files in an S3 bucket.](/docs/working-with-activities/list-of-available-activities/awsstorage/s3list)
* [Read an S3 file.](/docs/working-with-activities/list-of-available-activities/awsstorage/s3read)
* [Write data into an S3 file.](/docs/working-with-activities/list-of-available-activities/awsstorage/s3write)
* [Delete an S3 file.](/docs/working-with-activities/list-of-available-activities/awsstorage/s3delete)

You must [configure an S3 login resource](/docs/working-with-activities/list-of-available-activities/awsstorage/configuring-an-s3-amazon-web-services-resource) before you can start work on any of these activities.
