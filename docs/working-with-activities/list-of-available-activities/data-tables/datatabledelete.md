The **Data Table Delete** activity deletes records from a data table.

## Configuring the Data Table Delete Activity

To configure the **Data Table Delete** activity, fill in the required fields in the activity panel.

### Configuration

| Field | Required | Description | Example |
|-------|----------|-------------|---------|
| Name | Required | Specify the name for this activity. | MyActivity |
| Description | Optional | Specify the description for this activity. | Reads a file from S3 |
| Data Table | Required | Drag and drop a DataTable from the workspace. |  |

### Input

_Map the required input fields to provide data for this activity._

### Output

_The output fields are dynamically populated based on the activity result._
