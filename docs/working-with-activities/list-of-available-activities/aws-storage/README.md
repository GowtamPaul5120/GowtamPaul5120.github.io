---
title: "AWS Storage"
---


The AWSStorage activity group provides activities for interacting with Amazon Web Services (AWS) Simple Storage Service (S3). Use these activities to read, write, list, and delete files in S3 buckets, as well as configure S3 connections.

Before using the S3 activities, ensure you have configured an **S3 Connection** resource in your workspace with valid AWS credentials (Access Key ID and Secret Access Key).

## Available Activities

* [S3 Connection](S3Connection.md)
* [S3 Delete](S3Delete.md)
* [S3 File Trigger](S3FileTrigger.md)
* [S3 List](S3List.md)
* [S3 Read](S3Read.md)
* [S3 Write](S3Write.md)
