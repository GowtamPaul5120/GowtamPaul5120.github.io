# AzureStorage

Use AzureStorage activities to:

* [List all files in an ABS virtual directory.](/docs/working-with-activities/list-of-available-activities/azurestorage/listabsfile)
* [Read an ABS blob file.](/docs/working-with-activities/list-of-available-activities/azurestorage/readabsfile)
* [Delete an ABS blob file.](/docs/working-with-activities/list-of-available-activities/azurestorage/deleteabsfile)
* [Write data into an ABS blob file.](/docs/working-with-activities/list-of-available-activities/azurestorage/writeabsfile)

You must [configure an ABS login resource](/docs/working-with-activities/list-of-available-activities/azurestorage/configuring-an-abs-azure-blob-storage-resource) before you can start work on any of these activities.
