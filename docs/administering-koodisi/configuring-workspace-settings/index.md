title: "Configuring Workspace Settings"
sidebar_position: 1
---


To configure a workspace, navigate to it using the Workspaces drop-down list at the top-right corner of the page. Then, click the **Settings** icon in the main menu to view the Settings page.

Use the Workspace section of the Settings page to configure your workspace.

:::info
Only an administrator can create or configure workspaces.
:::

Workspace settings in Koodisi are organized into the following sections. Click on each to learn more:

- [workspace-general-settings.md](/docs/administering-koodisi/configuring-workspace-settings/workspace-general-settings)
- [managing-api-collections](/docs/working-with-workspaces/managing-workspace-level-entities/managing-api-collections/ "mention")
- Vaults: See [managing-secrets](/docs/working-with-workspaces/managing-workspace-level-entities/managing-secrets)
- [managing-keys-and-certificates](/docs/working-with-workspaces/managing-workspace-level-entities/managing-keys-and-certificates)
