---
title: "Creating Attribute Tags"
sidebar_position: 1
---


To create an attribute tag:

1. Navigate to the Settings > Organization > Attribute Tags page and click **Define Tag** at the bottom of the list of active attribute tags.\
   A new empty row appears.
2. Add the attribute and the description in the fields provided.
3. Click **Save**.

:::info
Once you create an attribute tag, you can't edit it. You can only deprecate the tag and create a new one.
:::

When you use attribute tags in your activities, they appear in the Attribute Tags tab of the configuration panel below the canvas. This enables you to view all the attribute tags used in a workflow in a single location.
