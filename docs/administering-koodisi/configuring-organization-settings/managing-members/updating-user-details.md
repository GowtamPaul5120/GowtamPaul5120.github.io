---
title: "Updating User Details"
sidebar_position: 2
---

To update a user's details:

1. Navigate to the Settings > Organization > Members page, click the **Actions** icon adjacent to the user, and select **Edit Member**.\
   The Edit User page appears.
2. You can update only the following details of a user:
   - First Name
   - Last Name
   - Role
   - Permissions

For details on managing a user's permissions, see [managing-member-permissions-at-the-organization-level.md](/docs/administering-koodisi/configuring-organization-settings/managing-members/managing-member-permissions-at-the-organization-level).
