---
title: "The Credential Manager"
---


The Credential Manager provides a secure place for you to store and manage your Git-related credentials. It also provides a Local Datastore, which enables administrators and authorized users to manage application data.

To access the Credential Manager, navigate to **Settings > Git Settings**.

* [Managing Credentials](/docs/administering-koodisi/configuring-source-control-settings/managing-credentials)
* [Managing Application Data](/docs/administering-koodisi/configuring-source-control-settings/managing-application-data)
