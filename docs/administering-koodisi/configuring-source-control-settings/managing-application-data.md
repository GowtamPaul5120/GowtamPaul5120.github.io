---
title: Managing Application Data
---


:::info
This is a very rarely used and highly restricted feature. We recommend only senior administrators be allowed access to this task.
:::

Use the Local Datastore (**Settings > Git Settings > Local Datastore**) to perform advanced troubleshooting and data handling.&#x20;

Click **Open Local Datastore** to access the tool. The Local Datastore appears as a side panel.

The Local Datastore enables you to:

- **Export local datastore data**: Click **Export lDB**.
- **Import local datastore data**: Click to view the Import lDB Data modal. Remember that importing data will overwrite all existing data irreversibly.
- **Clear the local datastore**: Click **Clear Local Datastore**, and click **Delete** in the warning message that appears.
