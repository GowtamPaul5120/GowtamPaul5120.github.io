---
title: "Using Git to Develop Applications"
---


Git integration enables you to manage your application’s source code in a Git repository. By enabling Git on your application, you can store your project locally in your browser (LocalDataStore) and push or pull changes to and from a remote repository.

This guide explains:

* [What Git-Enabled Applications Are](/docs/using-git-to-develop-applications/what-are-git-enabled-applications)
* [How you can create Git-enabled applications](/docs/using-git-to-develop-applications/creating-a-git-enabled-application)
* [How you can use Git with applications in Koodisi](/docs/using-git-to-develop-applications/using-git-with-applications)
* [The key differences between single and multi-module repositories](/docs/using-git-to-develop-applications/single-vs.-multi-module-repositories)
* [How you can clone Git applications](/docs/using-git-to-develop-applications/cloning-git-applications)
