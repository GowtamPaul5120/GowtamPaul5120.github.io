# V 1.1.13

## **New Features**

* **Git Capabilities**
  * Users can now clone a Git application directly within the platform.
  * Added support for creating both local and remote branches, enhancing version control flexibility.

## **Enhancements**

* **Array Functions** – Expanded functionality to improve data manipulation and processing efficiency.
* **Improved Error Validations for Mapping** – Enhanced validation mechanisms to provide clearer error messages and prevent incorrect mappings.
* **Conflict Resolution Support** – Users can now resolve conflicts within the platform, ensuring smoother collaboration and version management
