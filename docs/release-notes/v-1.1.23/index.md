**New and Improved Features**

1. **Profile Support**\
   Profiles can now be associated with **Batch** and **Messaging** integration patterns.
2. **useProfile Activity**\
   Introduced a new **useProfile** activity to allow workflows to explicitly select and apply a profile at runtime, making integrations more modular and environment-aware.
