## New and Improved Features

1.  ### **Schema Updates for Git-Enabled Apps**&#x20;

    Schema handling has been updated to store schema content directly in Git-enabled applications. This ensures branch-level consistency and true version control.**No Actions Required on your side** please refer to this [Schema migration guide.](https://docs.koodisi.com/~/changes/222/using-git-to-develop-applications/schema-guidelines)​
2. **Anonymous Array Name for Common Activities**\
   Cleaner activity configurations with support for anonymous array naming.
3. **OpenAPI Improvements**&#x20;
   * View linked OpenAPI details directly in the configuration panel.
   * Update and delete OpenAPI specs from the Registry.
4. **Persistent Developer Settings**\
   Developer settings now persist across sessions, improving continuity and productivity.
5. **Ternary Mapping Support**\
   Added support for ternary expressions in mapping logic, simplifying conditional transformations.
6. **Session Replay Configuration (Beta)**
   * Initial setup for session replay integration for capture user sessions
7. **Observability and Performance Enhancements**
   * Reduced unnecessary re-renders for faster responsiveness.
   * Phase 1 improvements to **Largest Render Time** and **Total Render Time**.

### Bug Fixes

1. **Audit Log Improvements**\
   Enhanced query parameters and date filtering for more precise log analysis.
