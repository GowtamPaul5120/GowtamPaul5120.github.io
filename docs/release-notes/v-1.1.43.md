# V 1.1.43

### New Features

#### Enhanced Execution Log Filtering

Refined status filter options in execution logs make it faster to pinpoint and review specific entries for troubleshooting.

#### DataTable Query — Advanced Sorting Controls

New sorting enhancements in **DataTableQuery** activities allow for precise ordering, making large datasets easier to navigate and analyze.

&#x20;

### Fixes and Improvements

#### Smarter Git Conversion

Files are now automatically adjusted for improved readability and structure during the “Convert to Git” process, ensuring clean and consistent commits.

#### Streamlined Deployment Views

Deployment listings now automatically exclude deleted applications, keeping environment overviews clean and relevant.

#### Faster OpenAPI Spec & Schema Discovery

Improved detection for OpenAPI specifications and related schemas enables quicker integration setup and reduces manual search effort.

#### Reliable Test Case Creation

Test cases can now be consistently created from both deployments and environments, streamlining validation workflows.

#### Rest Client Context Reset

Users can now remove an existing context and select a new one without recreating the entire configuration, allowing faster test iterations.
