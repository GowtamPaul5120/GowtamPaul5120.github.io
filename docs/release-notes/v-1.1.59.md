### **New and Improved Features**

**1. Activity Builder Enhancements**

Expanded support for template and review modes, improved usability, and added advanced configuration options for smoother activity creation workflows.

**2. Test Cases Updates**

* Test cases are now generated only when the workflow is saved, ensuring cleaner and more intentional test management.
* Existing test cases can be reused across multiple runs, reducing setup effort and avoiding duplication.
* **Note:** All earlier test cases will be cleared when this update goes live in production. Please recreate any required test cases after deployment.

**3. Publish Workflow Improvements**

Improved restore capabilities with built-in conflict resolution for more reliable recoveries during publishes.

**4. Triaging Agent Enhancements - Early Beta**

Introduced metrics query capabilities powered by GenAI to help generate dashboards and insights faster.

**5. Event Service Notifier - Early Beta**

Introduced real-time event notifications, improving visibility and allowing workflows to react instantly to system events.

**6. Unary Negative Expression Support**

Extended expression capabilities with full unary negative support for more expressive and flexible logic building.

**7. Manifest Enhancements & Tours**

Improved manifest behavior, including better synchronization during application rename and an upgraded onboarding tour experience.

**8. Activity & Schema Editor Updates (Theming)**

Upgraded visuals and interaction patterns for a more consistent and modern editing experience.

**9. Array Nesting Improvements**

Added comprehensive support for nested array input/output handling in activities.

### **Bug Fixes**

**1. Import & Clone Reliability**

Resolved issues where clone actions did not display errors or failed table references during import.

**2. Test Page Navigation**

Improved tab navigation for smoother testing workflows.
