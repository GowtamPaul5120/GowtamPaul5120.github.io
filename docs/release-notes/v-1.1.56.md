#### New and Improved Features

1. **Scheduled Messaging for Integration Patterns** \
   Integration workflows now support scheduled execution of **Send Message** activity. Users can configure message dispatch at specific times or intervals for automated communications and batch processes.

#### Bug Fixes and Improvements

1. **Mapper UI Enhancements for Conditionals**\
   Improved conditional mapping interface for better visibility, intuitive editing, and seamless navigation between branches during complex logic definitions.
2. **Activity Auto-Install Fix**\
   Resolved an issue with auto-install workflows to ensure consistent installation of required activities during first-time execution.
