### **Features**

* **Schema Editor Enhancements**: Users can now edit schema titles in the schema editor.
* **Lookup Enhancements**: Improved lookup functionality for better usability.
* **AI Schema Generation**: Users can now generate schemas using AI directly within the application.
* **Allow File Uploads in FormData**: Forms now support file uploads via `formdata`.
* **Render Markdown in AI Panel**: AI-generated content now supports Markdown formatting.

### **Enhancements**

* **Git Enhancements**:
  * Multi-module enhancements for Git support.
* **User Experience & UI Enhancements**:
  * Tooltip added for Output Schema fields.
  * Improved fetching of `GroupForm` from the backend.
* **Authentication & Security**:
  * **SSO Updates**: The "Reset Password" section is now hidden when SSO is enabled.
* **Refactors**:
  * Refactored `GroupForm` fetching mechanism.

### **Bug Fixes**

* Fixed Key Vault CSS issues.
* Fixed errors persisting in the panel after application deletion.
* Fixed editing default values in form fields.
