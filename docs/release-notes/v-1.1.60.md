## **New and Improved Features**

#### **Canvas & UI Enhancements**

* Refined theming across the flowboard for a cohesive, modern visual experience.
* Dynamic loader added across all screens for smoother transitions.
* Improved rule builder with enhanced dynamic color cues and simplified iconography.
* Enhanced import and invite modals featuring cleaner layouts, bulk email entry, and editable tokens.

#### **Event Notifier Enhancements**

* Now supports SFTP-based event triggers, broadening automation use cases.
* Redesigned configuration experience with support for external Config APIs.
* Improved dashboard usability for monitoring, managing, and validating notifier events.

#### **Import & Application Improvements**

* A unified artifact creation flow now supports generating required tables consistently across imports, templates, and Git-based applications.

#### **Observability Enhancements (Beta)**

* The deployments dashboard is now organization-scoped, giving users a unified, consolidated view of all deployments across the entire org.anization so users have a unified dashboard to view deployments across org

### **Bug Fixes**

1. **Reliable Resource Duplication**\
   Resource duplication now preserves structure and metadata consistently.
2. **Edge Splitting Defaults Corrected**\
   Default values now apply correctly when creating new edges.
3. **Data Table Column Rendering Fix**\
   Column widths now scale and align properly across screen sizes.
4. **Import Modal Tree Height Correction**\
   The modal height now adjusts smoothly, improving usability for large imports.
5. **Test Case Cleanup on App Deletion**\
   Test case folders now automatically delete with their parent application.
6. **Rule Panel Cleanup**\
   Visual alignment improved and redundant icons removed, resulting in a cleaner experience.
7. **Fixed-Width Format Resource State Handling**\
   Configuration state now updates and syncs accurately during setup.
