## New and Improved features

### 1. Format activity group

Designed to seamlessly convert data between various formats, enabling easy integration and processing across systems.

Includes

- **XMLToJSON**: Converts data from XML format to JSON format.

### 2. Compression activity group

The Compression Activity Group is designed to efficiently handle file compression and decompression tasks, enabling seamless data exchange and optimized storage across systems.

Includes these activities

- **Compress**: Combines multiple files into a single compressed archive to save space and facilitate easier transfer.
- **Uncompress**: Extracts individual files from a compressed archive, making them accessible for further processing.
