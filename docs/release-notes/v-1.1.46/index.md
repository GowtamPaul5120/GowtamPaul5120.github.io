### New and Improved Features

1. **AutoMapper Agent (Beta)**\
   Introducing AutoMapper, a data mapping agent that lets you define mappings using natural language.
2. **Documentation Agent (Beta)**\
   Inline documentation is now AI-powered—developers can access the help section and chat with AI using natural language.
3. **Git Usability Enhancements**\
   Improved Git workflows for smoother imports, replacements, and repository handling.
4. **Custom Context**\
   Organizations can now define custom context variables for use across workflows, improving reusability and standardization.
5. **Default Response Schema Extraction**\
   Automatically extracts default response schemas from OpenAPI specifications for faster setup.

### Bug Fixes

1. **Import Modal Usability**\
   Improved usability of the import modal for a smoother experience.
2. **Git Import Replace**\
   Resolved issues when replacing imports in Git-based workflows.
3. **OpenAPI Default Type Support**\
   Enhanced OpenAPI handling with support for the `default` type.
