**New Features**

* **Autosave for Configuration Changes:** Added as a preference in user settings, allowing users to enable autosave for configuration changes.
* **Enhanced Git Repository Validation:** Improved validation checks when creating applications with Git integration.
* **AI-Generated Schema Registry (Beta) :** Introducing AI-powered schema generation for the schema registry.
* **Import Report Feature:** Implemented a reporting feature for importing artifacts to improve messaging
* **Preference Tab for Mapper Navigation:** Added a dedicated preference tab for easier navigation within the mapper.

**Improvements & Enhancements**

* **Schema Editor UI update:** Refactored the schema editor for a better user experience.
* **Workspace Table  Enhancement:** Improved usability and performance of the workspace table and switcher.
* **Default Selected Application for Import:** Ensures the previously selected application remains chosen by default during import.
* **Added XML & Plaintext Support:** Expanded test request and response body types to include XML and plaintext formats.

**Bug Fixes**

* **Array-to-Array Automapping Fix:** Resolved an issue with incorrect automapping of array data.
* **Workflow Validation Error Display:** Fixed an issue where workflow validation errors were not showing correctly.
* **ForEach Relative Expression Validation:** Ensured proper validation for relative expressions in ForEach children.
* **Invalid Snackbar on Group Node Connection:** Addressed a bug where an invalid snackbar appeared when connecting group nodes.
