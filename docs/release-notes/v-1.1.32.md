## New and Improved Features

1. **Global Schema Locator**\
   Jump straight to any globally referenced schema from the **Input**, **Output**, or **App Schema** tree—no manual searching required.
2. **Manifest-Based App Configuration**\
   Applications can now declare configuration directly in `manifest.json`. Custom inputs set in Engage flow to the runtime automatically, supporting fully dynamic apps without additional code.
3. **Key-less Data Tables**\
   Create data tables without a primary-key column when flexible storage is preferred for integrations.
4. **FileCopy Context Support**\
   **FileCopy** now accepts context input, improving multitenant capability.&#x20;

## Bug Fixes

1. **Rename Propagation**\
   When you rename an activity or group, every reference across the workflow (mappings, conditions, and documentation) updates instantly.
2. **Git Conversion Documentation**\
   Application documentation now generates correctly when an existing app is converted to a Git-managed application.
