# V1.1.51

#### New and Improved Features

1. **Data Table Validation**\
   TTL validation now supports up to 36 hours, enabling more flexible data management.
2. **Activity Builder Enhancements**\
   Added support for context blocks, improving activity design and reusability.
3. **Profiles & Governance**\
   Profiles now display owner details for both profiles and workspaces, strengthening governance and accountability.
4. **Attribute Tagging in Conditional Flows**\
   Attribute tags now propagate across IF/Else nodes, ensuring consistent usage without duplication.
5. **Application Tree Performance**\
   Enhanced application tree rendering with virtualization for smoother navigation and improved scalability.

#### Bug Fixes

1. **Login & Session Scheduler**\
   Prevented premature API calls before user login, improving load time and stability.
2. **Context Management**\
   Refined delete modal for custom context definitions, making context handling more intuitive.
3. **OpenAPI Improvements**
   * Fixed tree node width and input field issues.
   * Added loading states during OpenAPI spec deletion.
   * Removed the OpenAPI feature flag for full availability.
4. **Global Variables**\
   Paths now display correctly when deleting referenced global variables.
