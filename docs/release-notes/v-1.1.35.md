## **New and Improved Features**

1. **Git Enhancement: Show changes**\
   Developers can now view staged and unstaged changes clearly within the Git integration panel, enabling better control over commits and code review.
2. **Execution Logs Panel Enhancement**\
   The execution logs panel now includes additional context and UI updates to improve readability and traceability of workflow execution steps.

## **Bug Fixes**

1. **Test Case Persistence Fix**\
   Resolved an issue where saved test cases, metrics, and traces were not retained across sessions. Tab state now persists as expected.
