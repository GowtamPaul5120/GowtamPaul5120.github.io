### New and Improved Features

#### 1. Send Message

Added **Include Profile** option, allowing messages to automatically use origin profile details for richer context and consistency.

#### 2. JSON to XML Activity

Enhanced support for seamless transformation from JSON to XML, improving interoperability across systems.

#### 3. Profile Logging

Profile-level logging is now available in lower environments (**Development**), enabling better visibility during development and testing.

#### 4. Output Attribute Enablement

Output attributes can now be selectively enabled, giving more control over exposed data and downstream mappings.

***

### Bug Fixes

#### Custom Activity Traces

Improved trace handling to ensure custom activity traces no longer propagate incorrectly into newly loaded workspaces.

#### Additional Fixes

* **Edit Activity** now consistently applies the correct default schema.
* **Nested Group Iterators** now resolve and apply schemas correctly.
* **Custom Activities** now reliably initialize with their default configuration.
* **Manifest Templates** now render with improved consistency.
* **Metrics Traces** now persist the selected default sort order across views.
