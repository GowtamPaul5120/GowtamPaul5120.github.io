### New and Improved Features

#### 1. Triaging Agent

Interact directly with your integrations using natural language. The triaging agent understands workflow context, classifies issues more accurately, and helps reduce investigation time across support and operations.

#### 2. Retry Capability

A new workflow resilience feature that automatically retries transient failures. This has significantly reduced support interventions by allowing workflows to recover without manual re-runs.

#### 3. Workflow Generation Improvements _(GenAI Feature)_

Automatically detects and generates schemas during workflow creation. Mappable fields are intelligently typed and added, accelerating build time and reducing manual configuration effort.

#### 4. Multiple Activity Generation with OpenAPI _(GenAI Feature)_

Generate multiple activities directly from OpenAPI schemas, enabling faster API integration setup and significantly reducing repetitive configuration work.

#### 5. Mapper & Builder Performance Upgrade

* Faster rendering and smoother schema extension
* Optimized TreeCore and expression evaluation
* Save workflow during schema import
* Improved schedule activity validation

#### 6. Git & Governance Enhancements

* Enforced branch naming policy for better repository hygiene
* Reliable variable updates on branch switch
