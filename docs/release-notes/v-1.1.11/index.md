## Enhancements

* **Auto Import Missing Groups**: Automatically imports missing groups when importing an application.
* **Save Metrics Filters**: Users can now save applied metrics filters for reuse.
* **Duplicate Git Applications**: Added functionality to allow users to duplicate Git applications.

## Fixes

* Added support for handling empty body in X-URL-Encoded content type.
* Resolved issue where logs were not visible in metrics.
