## New and Improved Features

### **Activity Builder Enhancements**

**Duplicate & Validation Support**\
The Activity Builder now detects duplicate configurations and provides inline validation feedback, improving authoring reliability.

**Condition Controls**\
You can now dynamically hide or disable conditions within activity configurations, enabling more responsive and flexible logic in complex workflows.

### **GenAI Improvements**

**Application Documentation Generator**\
Generate documentation for your workflows using GenAI — includes a structured summary of APIs, variables, and workflow logic to improve understanding and debugging.

**AI Support in Import Modal**\
The schema import modal now includes GenAI-powered assistance to simplify and accelerate schema configuration.

### **Initiate Batch: Multi-Extension Output Schema Support**

Output schema definitions now support multiple file extensions, allowing for broader compatibility and smoother integrations across systems.

### **Group Activity Enhancements**

Enhanced copy-paste support for activities inside group blocks for faster modular development.

### **Editor Improvements**

Markdown editor has been added to support better formatting and inline documentation within activities and groups.

### **FixedWidthFormat Activity**

New activity to support fixed-width file formats, where each field occupies a predefined number of characters.\
Enables transformations such as `JSONToFixedFormat`, allowing structured output without the need for delimiters.
