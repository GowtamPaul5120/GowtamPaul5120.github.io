#### New and Improved Features

1. **Git Operations**\
   Enhanced `useGitOperations` hook to manage selected tree node state, improving repository navigation and workflow efficiency.
2. **Plugin Management**\
   `PluginCard` now conditionally renders edit and delete options for artifacts, streamlining plugin management.
3. **Git Usability Enhancements**\
   Multiple improvements to Git workflows, including better credential recognition and support for multi-module setups.

#### Bug Fixes

1. **Schema Editor**\
   Save action is now correctly enabled after applying transformations.
2. **Collaboration**\
   Updated permissions for creating teams in collaborative mode.
3. **OpenAPI Registry**\
   Fixed issue with previewing OpenAPI documents in the registry.
