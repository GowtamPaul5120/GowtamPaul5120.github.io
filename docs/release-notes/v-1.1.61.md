## New Features

1. Encryption Resource Improvement&#x20;

Introduced environment-specific certificates. A single certificate resource can now hold multiple certificates, scoped per environment. • Automatic certificate rollover: when a certificate expires, the system seamlessly switches to the next valid certificate.

1. Mapping Subflow Context Improved context handling during initial data population to ensure mappings load and execute reliably across subflows.
