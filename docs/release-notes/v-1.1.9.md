#### New Features

* **Mapper Enhancements:**
  * Added support for `-` in headers and parameters (hyphen is allowed only between words).
  * Added support for `current()` property in mapping expressions.
  * Ensure subtraction operations are used with spaces for proper mapping.
* **Metrics Enhancement:**
  * Added ability to mask lookup column data, ensuring it appears as masked data in metrics.
* **SFTP Resource Update:**
  * Introduced SSH key definition and usage support.
