## New and Improved features

### 1. Attribute tag search with metrics

This update allows users to assign searchable attributes to specific fields within workflows. Enabling efficient querying of traces based on these attributes. ([USER STORY 1533](https://dev.azure.com/ivoyant/PNX/_workitems/edit/1533))

### 2. [Azure Storage Group](../working-with-activities/#azure-storage-group) - S3 Connector

This integration enhances cross-cloud file management and simplifies your workflows within the Azure Storage Group. ([USER STORY 1763](https://dev.azure.com/ivoyant/PNX/_workitems/edit/1763))

Includes these activities

- **Read Files:** Retrieve files from Amazon S3 buckets effortlessly.
- **Write Files:** Upload and store data to S3 buckets seamlessly.
- **List Files:** Access and manage the file list in specified S3 buckets.
- **Delete Files:** Easily remove files from your S3 buckets.
