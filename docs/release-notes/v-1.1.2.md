## New and Improved features

### 1. PGP encryption capabilities

This update introduces support for PGP encryption, enabling secure data encryption and decryption. It ensures confidentiality and integrity by leveraging public and private key cryptography for protecting sensitive information during transmission or storage. ([USER STORY 2768](https://dev.azure.com/ivoyant/PNX/_workitems/edit/2768))

### 2. Format activity group

Designed to seamlessly convert data between various formats, enabling easy integration and processing across systems.&#x20;

Includes these activities

* **CSVToJSON**: Converts data from CSV format to JSON format.
* **JSONToCSV**: Converts data from JSON format to CSV format.
