# V 1.1.8

#### New Features

* **Deployment Profiles:** Introduced to streamline deployment processes.
* **Client & Deployment-Scoped Variables:** Enhances deployment workflows by enabling environment-specific configurations.
* **Enhanced Template Management:** Users can now overwrite published templates and highlight featured templates by adding them to the homepage for quick and easy access.

Enhancements

* **User-Level SSO Configuration:** Allows admins to enable SSO customization, giving users the flexibility to opt in for custom login updates.
* **Import Flow Enhancement:** Users can now rename imports during the import process.
* **Smart Deploy Optimization:** MD5 hashing minimizes unnecessary updates, reducing deployment time.

Fixes

* Resolved issue where tests were getting lost upon saving.
* Fixed issue where saving a test with `form-urlencoded` content type reset it to JSON.
* Ensured consistency in Activity Template casing.
* Addressed issues with new member invitations.
