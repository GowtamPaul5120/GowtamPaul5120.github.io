## New and Improved Features

1. Inline AI Mapper Redesign The mapping experience has been modernized with a redesigned inline mapper, making navigation, field selection, and transformations faster and more intuitive.
2. Inner Group Nested Schema Generation Workflows now generate nested schemas for inner groups, with accumulated outputs from outer loops automatically available in the End activity. This makes it easier to work with iterative and grouped logic while preserving complete execution results.
3. URL Authorization Support URL-based authorization is now supported, enabling secure access patterns for integrations that require authenticated endpoints.
4. Improved Search and Trace Filtering Search capabilities now include advanced operators like CONTAINS and NOT CONTAINS, allowing faster and more precise trace analysis.
5. UI and Theming Updates The header toolbar, activity creation screens, and overall theming have been refined to deliver a cleaner, more consistent visual experience.
6. Enhanced Date Range Picker The date range picker has been improved for better usability and accuracy when filtering logs, traces, and workflow activity.

## Bug Fixes

1. Output Schema Handling for Array Roots Output schema transformations now correctly handle array root schemas, ensuring reliable mappings for complex payloads.
2. Workflow Logging Tree Visualization The workflow logging tree now renders with correct height and structure, improving readability during debugging and monitoring.
3. Safe Handling of Discarded Changes Discarding unstaged changes no longer risks unintended data loss, making editing and experimentation safer.
4. Validation and Test Flow Improvements Special character validation and test example flows have been strengthened, resulting in more predictable behavior during development and testing.
5. Consistent Activity Fetching Improvements Improved validations for consistent activity fetching and added detailed logging for activity config retrieval failures.
