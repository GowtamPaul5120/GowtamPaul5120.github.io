#### New and Improved Features

1. **Auto-Install of Activities**\
   Activities can now be automatically installed and updated, streamlining setup and reducing manual steps.\
   Use the **Sync Activity** button in the footer to fetch the latest activity versions manually. This also runs automatically every hour to keep your workspace up to date.\
   Please use the **Review Flow** to complete and validate activities before deployment.\
   &#x20;
2. **Mapper UI Update**\
   Redesigned Mapper interface for a smoother experience, improved visual clarity, and faster mapping updates.
3. **Git Enhancements**\
   Git integration now includes an interactive **Commit Graph** and additional improvements for better repository visibility and commit tracking.

#### Bug Fixes

1. **Workflow Config – Fallout Enable Switch**\
   Added the missing Fallout Enable Switch option in Workflow Configuration for improved workflow control.
