## Advanced

| Field       | Required | Description                                                                     | Example       |
| ----------- | -------- | ------------------------------------------------------------------------------- | ------------- |
| Sort By     | Optional | Sorts the result list by selected criteria: Name, Size, Date Modified, or Path. | Date Modified |
| Max Results | Optional | Limits the number of files returned in the result set.                          | 10            |
| Descending  | Optional | If selected, sorts the results in descending order.                             | Deselected    |
| Recursive   | Optional | If selected, includes files from subdirectories.                                | Deselected    |
